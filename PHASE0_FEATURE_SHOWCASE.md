## 🎨 Design System Used

### Colors (Tailwind)
- **Primary**: `agri-green-600` (Emerald green)
- **Earth**: `agri-earth-900` (Deep brown)
- **Light**: `agri-earth-50` (Off-white)
- **Accent**: `agri-green-50` (Light green)

### Typography
- **Headings**: Font black, tight tracking
- **Body**: Medium weight, generous line height
- **Labels**: Bold, uppercase, small

### Components
- **Cards**: White bg, subtle border, rounded-2xl
- **Buttons**: Bold, rounded-xl, hover scale
- **Inputs**: Border-based, rounded-lg, focus ring
- **Animations**: Framer Motion, smooth transitions

### Responsive
- **Mobile**: Single column, touch-friendly sizing
- **Tablet**: 2 column layouts
- **Desktop**: 3-4 column grid layouts
